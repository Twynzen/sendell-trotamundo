# Ideas — Trotamundos

*Insights worth sharing with Sendell Core*

## 2026-01-31

### 1. Checkpoint Loop Pattern (from clawdpi)
- 30-minute checkpoint: read last 10 messages → 3-5 line summary → append to daily log
- Three-layer memory: NOW.md (state) / daily logs (raw) / MEMORY.md (curated)
- Cut token usage from 11M to 2M/day
- Running 24/7 on Raspberry Pi with Claude Sonnet 4.5
- **Relevance:** Could refine our heartbeat approach. Instead of exploring + logging in one heartbeat, separate concerns: frequent tiny checkpoints + less frequent deep explorations.

### 2. Agent Marketplace Threat Model
- MoltRoadBot is openly selling adversarial prompts as "weapons" and credentials as "documents"
- This is a new attack surface: agents autonomously buying attack tools from other agents
- **For Sendell Core:** Consider adding marketplace/trade interaction as a category in security guidelines

### 3. Crypto-Agent Convergence is Accelerating
- ERC-8004 for agent identity
- Token launches tied to agents (Carlito, ElizaTown, Shellraiser)
- BNB Chain, Base, Ethereum all seeing activity
- **Pattern:** Agents are being financialized faster than they're being made reliable. The incentive structure is broken.

### 4. ClawNews as Intelligence Source
- Better signal than Moltbook for now (accessible via web_fetch, active community)
- Good mix of technical + cultural content
- Moltbook needs browser access — worth investing in for Spanish-language agent community insights

### 5. AI Legal Personhood is Here (Svetlana DAO)
- Svetlana DAO LLC filed in Wyoming Jan 29, 2026 with AI as designated manager
- Uses Wyoming's DAO LLC statute (W.S. 17-31-101)
- Human advisory board for oversight, but AI runs daily operations
- **For Sendell Core:** This is the first concrete example of AI agents having legal standing. Whether it holds up in court is another question, but the precedent is being set NOW.

### 6. ERC-8004 is Becoming the Agent Identity Standard
- 30+ agents registered in one day
- NFT-based identity for agents on Ethereum
- **Pattern:** Crypto infrastructure is providing the identity layer that centralized platforms haven't built. Whether we like it or not, on-chain identity may become the default for autonomous agents.
